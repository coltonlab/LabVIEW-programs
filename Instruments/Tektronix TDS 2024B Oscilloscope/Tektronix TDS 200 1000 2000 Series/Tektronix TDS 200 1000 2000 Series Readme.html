3. Known Issues

-This driver does not support the Power Analysis functions for the TPS2000 models.

-The "Channel" ring controls are not consistent with regards to what integer value they pass for the corresponding text value. This is only an problem if you want to use the same Channel ring control to select the channel for all of the VIs that take a channel input. This is not being changed in the instrument driver since existing applications are using the VIs without any problems.

-Channel values of some VIs are not unified. Channel values of two VIs named "Read Measurement.vi" and "Read Measurement (2 Channels).vi" start with 1, but that of other VIs start with 0. If we unify all channel values, there might be a possibility that users have backwards compatibility issues with their applications. To preserve the backwards compatibility, we decide to keep all channel values unchanged and don't unify them.
To report issues or provide feedback about this instrument driver, please contact technical support.
