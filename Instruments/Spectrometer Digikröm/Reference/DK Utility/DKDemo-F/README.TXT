README.TXT for VB CM11X Demo file.

The CMDemoBZ.EXE is a compress executable file, which consists of the setup
program disk 1 of 2 and disk 2 of 2. There is another way to save the setup
file in two floppy disks: disk 1 and 2.

The source code is stored in CMDemoBS.EXE, which is also a compressed
executable file.
